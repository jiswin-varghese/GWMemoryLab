# Gravitational-Wave Phase Evolution

## Objective

Compute the gravitational-wave phase from the frequency evolution.

## Theory

The gravitational-wave frequency is related to the phase by

dφ/dt = 2πf

Integrating over time gives

φ(t) = ∫2πf(t)dt

## Numerical Method

Euler integration
